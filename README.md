# Proyecto Django sistemaAyM

Este proyecto fue creado manualmente debido a problemas con la generación automática.

## Estructura
- manage.py
- sistemaAyM/
  - settings.py
  - urls.py
  - wsgi.py
  - asgi.py

## Cómo iniciar el servidor

1. Instala las dependencias si no lo has hecho:
   ```
pip install django
   ```
2. Ejecuta el servidor:
   ```
python manage.py runserver
   ```

## Configuración
- El archivo settings.py contiene la configuración principal.
- El archivo urls.py gestiona las rutas.
