# Manual de Usuario - Sistema AyM

## Funcionalidades avanzadas

### 1. Filtros avanzados y exportación a Excel

- En los módulos de Proveedores, Inventario y Órdenes de Compra, puedes filtrar los registros usando los campos de búsqueda y selección de estado.
- Para exportar los resultados filtrados a Excel, haz clic en el botón "Exportar a Excel". El archivo descargado contendrá solo los datos filtrados.

### 2. Dashboard y reportes

- El dashboard de proveedores muestra gráficos de estados y promedios de scorecard.
- Las alertas de facturas vencidas e incidentes críticos aparecen automáticamente en la parte superior de la lista de proveedores.

### 3. Acciones rápidas

- Usa los botones de acción en cada fila para ver el reporte detallado, editar o eliminar registros.
- En órdenes de compra, puedes filtrar por estado y buscar por código o proveedor.

### 4. Automatización de inventario

- El sistema sugiere automáticamente la reposición de artículos cuando el stock está por debajo del mínimo configurado.
- Al recibir una orden de compra, el stock se actualiza automáticamente.

### 5. Requisitos

- Navegador moderno (Chrome, Edge, Firefox)
- Acceso autorizado al sistema
